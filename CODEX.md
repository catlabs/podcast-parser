# Codex entrypoint

Use `AGENTS.md` first. It points to the shared `.ai/` source of truth used by
both Codex and Claude Code.

When starting a role:

- Mentor: follow `.ai/agents/ai-mentor.md`.
- Coder: follow `.ai/agents/coder.md`.
- Operator: follow `.ai/agents/operator.md`.

Keep Codex-specific guidance thin. Shared workflow, Git rules, memory, reports,
and approval constraints belong in `.ai/`.
